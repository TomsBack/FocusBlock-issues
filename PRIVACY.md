# FocusBlock Privacy Policy

Last updated: June 10, 2026

FocusBlock does not collect, sell, share, transmit, or store personal data outside your browser.

FocusBlock runs locally in Chrome to apply a friction countdown overlay on user-selected distracting sites.

## Data Stored Locally

FocusBlock stores blocked site rules, settings, schedules, snooze state, and local reminder state in Chrome extension storage. This data stays in your browser unless you use browser sync features controlled by Chrome.

## Page Interaction

FocusBlock uses host access and content scripts so it can detect user-defined blocked sites and display the local countdown overlay on matching pages. It does not transmit page content or browsing data.

## Data Collection

FocusBlock does not use analytics, advertising SDKs, external tracking, remote logging, or external servers. It does not transmit blocked-site lists, browsing history, page content, or settings to the developer or any third party.

## Permissions

FocusBlock requests browser permissions only to provide local focus-blocking features: storage for settings, tab/navigation access for detecting relevant pages, host access for showing the overlay, context menus and shortcuts for controls, alarms for schedules, and notifications for local reminders.
